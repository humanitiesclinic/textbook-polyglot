Definition 2.1.2.

Phonologically-based systems represent sounds:
🔗
  • Alphabetic scripts represent individual phonemes (e.g., Latin alphabet: a, b, c...)
    🔗
  • Abjads represent consonants but not vowels (e.g., Hebrew, Arabic)
    🔗
  • Abugidas represent consonants with attached vowel marks (e.g., Devanagari, Thai, Khmer)
    🔗
  • Syllabic scripts represent whole syllables (e.g., Japanese hiragana, Cherokee)
    🔗
Morphologically-based systems represent meaningful units:
🔗
  • Logographic scripts represent words or morphemes (e.g., Chinese characters, Egyptian hieroglyphics)
    🔗
